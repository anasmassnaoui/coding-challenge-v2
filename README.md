# coding-challenge-v2
A simple reviews app collector

## Install
``` bash
    npm install
    cd client
    npm install
```

## Run
``` bash
    # to run backend
    npm start
    # to run front end
    cd client
    npm start
```

## Visite

- go to http://127.0.0.1/ to visite MVP
- go to http://127.0.0.1:3000/ to visite V2

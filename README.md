# coding-challenge-v2
A simple reviews app collector

## Install
``` bash
    npm install
    cd client
    npm install
```

## Run
``` bash
    # to run backend
    npm start
    # to run front end
    cd client
    npm start
```

## Visit

- go to http://127.0.0.1:8080/ to visit MVP
- go to http://127.0.0.1:3000/ to visit V2
